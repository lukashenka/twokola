Search Engine Friendly urls for Wordpress

== Description ==

Main features:
1. Change or remove category url base.
2. Change archives url.
3. Change tags url base.
4. Change pages url ex. http://yoursite.com/pagename.html
5. Change aurthor page url.
6. Change posts url.

== Installation ==

1. Upload `wp-sef-urls` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "Settings | WP SEF urls" administration page to change settings.
