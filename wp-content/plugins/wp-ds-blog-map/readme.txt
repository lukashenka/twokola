== Description ==

This plugin generates tag cloud, posts list ordered by categories and tree of the pages. The list may be placed in any part of wordpress page contents

== Installation ==

Download: <a href="http://wp-plugins.diamondsteel.ru/">WP DS Blog Map</a>

1. Upload the FOLDER 'wp-ds-blogmap' to the '/wp-content/plugins/'
2. Activate the plugin 'WP DS Blog Map' through the 'Plugins' menu in admin
3. Go to 'Options' or 'Settings' and then 'DS Blog Map' to change the options

= Usage =

1. Choose the page or post where you want DS Blog Map to be displayed.
2. Place the <code>[postlist]</code> string in any part of the page and save/publish your page.

That's all. The replace string may variate through plugin options.
